# Contains the following endpoints:

## All Endpoints are protected by JWT Authentication.

*************************************************
## 1. Food Endpoints:
### 1.1. Get all Food:
#### GET /food
### 1.2. Get Food by ID:
#### GET /food/{id}
### 1.3. Add Food:
#### POST /food
### 1.4. Update Food:
#### PUT /food/{id}
### 1.5. Delete Food:
#### DELETE /food/{id}
*************************************************
